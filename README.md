# Sniffer-Bot
Телеграм бот для отслеживания изменений в профиле пользователей.

С помощью этого бота вы можете отслеживать все изменения в профиле пользователей.<br>
Для его работы вам потребуется сессия телеграм аккаунта, которая и будет этим заниматься.

## Запуск
### <b>В файле</b> [data/constants.py](https://github.com/boostNT/Sniffer-Bot/blob/main/data/constants.py):<br><br>
- Получите токен бота у <b>@BotFather</b> и вставьте его в переменную <b>TOKEN</b>.

- При необходимости получите данные для сессии (<b>api_id, api_hash</b>) на [<b>этом сайте</b>](https://my.telegram.org/auth) и замените их на свои, или просто положите файл сессии в корень папки бота (ее название должно быть <b><i>main</i></b>).<br>

- Внесите айди пользователей для вайт листа в переменную ADMINS, с которыми будет взаимодействовать бот.
